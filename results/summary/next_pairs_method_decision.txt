RNA Method Decision for Pair 2 and Beyond
=========================================

Selected method: limma-voom

Decision rationale (based on first pair CMP vs Erythroblast):
- limma-voom and DESeq2 are highly concordant (4209 shared DE genes with 100% direction agreement).
- limma-voom detected slightly more significant genes (5057 vs 4799) while preserving the same core biology.
- limma-voom is computationally lighter for repeated pairwise runs, which helps finish additional required pairs on schedule.

How to apply this decision:
- Keep both methods in first-pair writeup (requirement satisfied).
- For next assigned pairwise comparisons, run limma-voom as default RNA method.
- If a new pair has unstable QC or contradictory biology, run DESeq2 as sensitivity check and document the exception.

Reusable command template:
Rscript scripts/rna_seq/run_pairwise_rna_generic.R --cell_a HSC --cell_b CMP --method limma_voom --out_dir results/limma_voom/pairwise_hsc_vs_cmp
